## 🚀 How to Use

1. **Open the website**: Simply open `index.html` in a web browser
2. **Navigate sections**: Use the sticky navigation bar to jump between sections
3. **Filter vehicles**: Click filter buttons to view specific vehicle categories
4. **View details**: Click "Quick View" or "Details" to see vehicle information
5. **Schedule test drive**: Use the test drive button in the modal or contact form
6. **Contact**: Fill out the contact form or use provided contact details
